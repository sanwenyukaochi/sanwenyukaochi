# 手绘风食物注解提示词记录

## 适用场景

用于给美食照片添加白色手绘风注解，例如火锅、饮料、小菜、蔬菜、配菜、餐桌氛围等。整体效果类似“日系可爱手账 / 美食日记涂鸦”。

## 完整提示词

```text
请观察照片中的元素，并为每个物件加上有意义的手绘风注解。

【描写规则】
- 使用像白色笔画的细线手绘线条。
- 一笔画风格、随性、略带不均匀感。
- 沿着物件外围加上描边轮廓。
- 用箭头、虚线或虚线引导线做出视线引导。
- 保留原始照片的构图、颜色、光线和真实物件，不要替换场景，只在照片上叠加注解。

【文字规则】
- 使用手写风格字体，日系可爱感。
- 句子简短，像自言自语的小碎念。
- 语气偏日记感，带一点情绪。
- 中文文字要自然、可爱、轻松。

【注解生成规则】
- 饮料：描述味道、温度、心情，例如：清爽、微甜、刚刚好。
- 食物：描述口感、好吃程度，例如：松软、超好吃、香到不行。
- 空间：描述氛围，例如：很放松、喜欢这种感觉。
- 整体：加入一句总结，例如：今天有点幸福～。

【装饰规则】
- 适度加入热气、闪光、爱心、星星、小表情等元素。
- 不要过度装饰，保留空白空间。
- 注解不要遮挡主体食物太多，尽量放在照片空白处。

【本图物品注解示例】
1. 中间的大火锅：描边并标注“麻麻辣辣，香到不行～”。
2. 火锅中心翻滚的汤底 / 热气：添加热气涂鸦并标注“咕嘟咕嘟，看着就饿了～”。
3. 左侧饮料罐：描边并标注“冰冰凉凉，清爽刚刚好～”。
4. 上方绿色小菜盘：描边并标注“脆脆的，先开胃一下～”。
5. 右上方菌菇 / 豆制品 / 瓜片类配菜盘：描边并标注“这一盘下锅会很鲜～”。
6. 右侧娃娃菜 / 白菜盘：描边并标注“清甜解辣，安心加分～”。
7. 右下方丸子 / 豆泡 / 小香肠拼盘：描边并标注“每样都想吃，超满足～”。
8. 餐桌空间氛围：在不显眼的位置标注“围着热锅坐着，感觉很放松～”。
9. 整体总结：在合适空白处写“今天有点幸福～”。

整体感觉要像一页个人美食日记：温暖、放松、有一点情绪、可爱、热闹，但不要乱。
```

```text
请给照片里的【主体食物】、【饮料】、【配菜】、【餐桌空间】添加白色手绘风注解。

主体食物注解：【例如：麻麻辣辣，香到不行～】
饮料注解：【例如：冰冰凉凉，清爽刚刚好～】
配菜注解：【例如：清甜解辣，安心加分～】
空间氛围注解：【例如：围着热锅坐着，感觉很放松～】
整体总结：【例如：今天有点幸福～】
```
